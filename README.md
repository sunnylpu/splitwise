# Run and deploy your Splitwise app

This contains everything you need to run your app locally.

## AI Used
- **Google Gemini** (Advanced Agentic AI Assistant)

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env` to your Gemini API key from [Google AI Studio](https://aistudio.google.com/apikey).
3. Set up your Firebase project and replace the credentials in `firebase-applet-config.json`.
4. Deploy the `firestore.rules` to your Firebase Console.
5. Run the app:
   `npm run dev`
